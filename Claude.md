# Claude.md

## Project
Elite Syrian Cafe Website

## Objective
Build a premium, story-led cafe website for a Sydney-based cafe owner with Syrian heritage and an Armenian family connection. The site must feel elite, refined, warm, and culturally rich, not like a generic cafe template.

## Core Brand Truth
**We serve the food we truly love to eat ourselves.**

## Brand Context
- The owner is Australian-born with Syrian cultural roots.
- Her husband is Armenian and Sydney-raised.
- The cafe should reflect heritage, family, hard work, and quiet pride.
- The owner uses special ingredients sourced from Jordan.
- The brand should feel authentic, elevated, and emotionally real.

## Product Goal
Create a website that feels like a premium boutique hospitality brand.
The website should balance:
- modern Sydney cafe aesthetic
- Levantine warmth
- editorial storytelling
- understated luxury

## Design Direction
### Look and Feel
- elite
- premium
- elegant
- warm
- minimal
- editorial
- handcrafted
- culturally textured, but subtle

### Avoid
- generic cafe templates
- loud colours
- cluttered sections
- cheesy cultural motifs
- stock-template feel
- overexplaining heritage in a forced way

## Visual Style
### Colour Palette
- Cream: `#F6F0E7`
- Sand: `#DCC7A6`
- Olive: `#7A8363`
- Terracotta: `#B76A45`
- Deep Brown: `#3D2C24`

### Typography
- Headings: elegant serif
- Body: clean modern sans-serif
- Generous whitespace
- Strong visual hierarchy

## Tone of Voice
- warm
- refined
- quietly confident
- personal
- premium
- human

Do not sound corporate or over-marketed.

## UX Requirements
- mobile-first
- elegant scrolling experience
- lightweight and fast
- obvious CTA paths to menu and location
- excellent readability
- accessible colour contrast

## Technical Preferences
- Next.js
- Tailwind CSS
- reusable components
- responsive layout
- optimized images
- good SEO structure
- LocalBusiness schema
- clean code and folder structure

## Performance
- lighthouse target above 90 where practical
- lazy load gallery
- optimize for mobile
- compress images and use modern formats

## Implementation Notes for Claude Code
- Prioritise polish over unnecessary complexity.
- Keep sections visually rich but restrained.
- Use premium spacing and typography scale.
- Make the homepage the strongest page.
- Ensure the site feels like a destination brand, not a basic local cafe website.
- Build with placeholder content and reusable sections so content can be swapped later.
- Use realistic placeholder images or image slots if assets are unavailable.
- Keep the codebase neat and production-friendly.

## Final Standard
The finished site should feel like:
**a high-end Sydney cafe brand with Levantine soul, elegant storytelling, and premium visual identity.**
